# Film Arama Uygulaması

Bu proje, kullanıcıların film araması yapabileceği basit bir web uygulamasıdır. OMDb API kullanarak film bilgilerini getirir ve kullanıcıya gösterir.

## Proje Yapısı

### Dosyalar ve Klasörler

- **Img/**: Uygulamada kullanılan resimler.
- **Index.html**: Ana HTML dosyası.
- **Scripts/Index.js**: JavaScript kodlarını içeren dosya.
- **Style/Index.css**: CSS stil dosyası.

## Kullanılan Teknolojiler

- HTML
- CSS
- JavaScript (jQuery)
- OMDb API

## Kurulum ve Kullanım

1. Bu projeyi klonlayın veya indirin.
2. Proje dizinine gidin ve `Index.html` dosyasını bir tarayıcıda açın.

### Örnek Kullanım

1. Arama kutusuna bir film adı girin.
2. "Ara" butonuna tıklayın veya Enter tuşuna basın.
3. Arama sonuçları ekranda görüntülenecektir.

## OMDb API Anahtarı

Proje içerisinde OMDb API kullanımı için bir API anahtarı gerekmektedir. `Scripts/Index.js` dosyasındaki `apiKey` değişkenine kendi OMDb API anahtarınızı ekleyin.

const apiKey = "YOUR_OMDB_API_KEY"; // Kendi OMDb API anahtarınızı buraya yazın.
